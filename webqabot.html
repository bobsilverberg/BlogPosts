We are pleased to announce that our brand new IRC bot, webqabot, has joined our IRC channel. webqabot is very similar to mozwebqabot, who used to inhabit the channel, except it is built upon Hubot. It does most of what mozwebqabot did, and also has a nifty new feature which allows it to announce activity in any of our GitHub repositories. You may therefore notice a small increase in messages from the bot in the channel whenever GitHub issues and pull requests are opened and closed, as well as when code is committed. There are a few issues open for webqabot, so if you're interested in helping out, and getting some experience with node.js and CoffeeScript, check them out. You can find more details on webqabot in its README file.
